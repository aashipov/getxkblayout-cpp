# X11 keyboard layout (C++) #

## Why ##

Keyboard layout indicator for a Window Manager panel (e.g., `tint2`)

## Requirements ##

`gcc` with `ld` or `clang` with `lld`, `cmake` with `make` or `meson` with `ninja`

## Build script ##

`release.sh` or `debug.sh`
